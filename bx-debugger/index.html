bx-debugger is an experimental NodeJS-based debugger that reads warnings from a transpiled .batx file and formats them, making finding them and fixing them faster. The debugger includes the line number where error is located and also will eventually display a help screen for that specific error. Currently the version is 1.0b and can be found on GitHub. .
